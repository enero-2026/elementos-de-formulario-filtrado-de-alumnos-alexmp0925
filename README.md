# Aplicación de Gestión de Estudiantes

Esta aplicación React Native proporciona una interfaz simple para gestionar y buscar registros de estudiantes.

## Características

- Buscar estudiantes por nombre o matrícula
- Mostrar lista de estudiantes con información de matrícula
- Navegación por pestañas para diferentes secciones

## Detalles de Implementación

1. Componente TextInput integrado de React Native Paper para funcionalidad de búsqueda
2. Gestión de estado para consulta de búsqueda
3. Lógica de filtrado aplicada al arreglo de estudiantes usando el método filter de JavaScript
4. Actualizaciones en tiempo real de la lista de estudiantes basadas en el campo de búsqueda

## Tecnologías Utilizadas

- React Native
- Expo
- React Native Paper
- Expo Router
